Web čitači

Web čitači je aplikacija koja je namenjena prikazu sadržaja web stranica. Stranice se mogu dobivati kako iz fajl-sistema, tako i putem TCP/IP mreze. Protokol kojim komuniciraju web čitač i web server je HTTP protokol. Danas su najpoznatiji sledeći web čitači:

  1. Microsoft Internet Explorer (http://www.microsoft.com/ie)
  2. Mozila (http://www.mozilla.org)
  3. Mozila (http://www.opera.com)
