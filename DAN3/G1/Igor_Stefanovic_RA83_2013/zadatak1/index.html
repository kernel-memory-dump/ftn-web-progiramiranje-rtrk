Web čitači

Web čitači je aplikacija koja je namenjena prikazu web stranica.
Stranice se mogu dobaviti kako iz fajl-sistema, tako i putem TCP/IP mreze.
Protokol kojim komuniciraju web čitač i web server je HTTP protokol. Danas su
najpoznatiji sledeći web čitači:

  1. Microsoft Internet Explorer(http://www.microsoft.com/ie)
  2. Mozilla(http://www.mozilla.org)
  3. Opera(http://www.opera.com)

Obratite pažnju da su svi nazivi web čitača u gornjoj listi linkovi koju upućuju na
adrese navedene u zagradi!
