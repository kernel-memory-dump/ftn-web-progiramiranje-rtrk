Web čitači

Web čitač je aplikacija koja je namenjena prikazu sadržaja web stranica. Stranice se mogu dobaviti kako iz fajl-sistema, tako i putem TCP/IP mreže. Protokol kojim komuniciraju web čitač i web server http protokol. Danas su najpoznatiji sledeći web čitači

1. Microsoft Internet Explorer (http://microsoft.com/ie)
2. Mozila (http://mozilla.org)
3. Opera (http://www.opera.com)
